# API Ruby and with Html

**Framework's**
- *TailwindCSS*
- *Particles.js*

**Routes**
- `/register`
- `/login`

## How to use

```bash
git clone https://github.com/171ntw/api-ruby.git
cd api-ruby
```

### Install the dependencies

```bash
gem install sinatra json bcrypt
```
**or**
```bash
bundle install
```

### Start the Server

```bash
ruby main.rb
```
